## Fibonacci series of N numbers

### Algorithm

*Step 01* : **Start**\
*Step 02* : **Read num**\
*Step 03* : **Set f= 0, s= 1 & i= 2**\
*Step 04* : **Print f & s**\
*Step 05* : **while i<num, repeat steps 6 to 9**\
*Step 06* : **Set t= f+s**\
*Step 07* : **Print t**\
*Step 08* : **Update f= s & s= t**\
*Step 09* : **Increment i by 1**\
*Step 10* : **Stop**
