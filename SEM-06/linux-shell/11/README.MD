## Reverse of a number

### Algorithm

*Step 01* : **Start**\
*Step 02* : **Read num**\
*Step 03* : **Set rev= 0**\
*Step 04* : **while num > 0, repeat steps 5 to 7**\
*Step 05* : **Set dig= num % 10**\
*Step 06* : **Update num= num / 10**\
*Step 07* : **Update rev= (rev*10) + dig**\
*Step 08* : **Print rev**\
*Step 09* : **Stop**
